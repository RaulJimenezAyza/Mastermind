The Game:

Objective: Discover the secret code by trying different color combinations

How to Play: A four-color code containing the buttons is displayed. The code can contain multiples of the same color. The game is about trying to guess the correct code (color and order) by clicking on the colored buttons and then pressing the submit button (green tik). Your guess will be compared to the code. For each correct color that is in the correct position, a black peg will be displayed. However, for each correct color that is in the wrong position, a white peg will be displayed. Use this information to determine the correct code.
